# VPS DevUs

Repositorio operativo para documentar y mantener la migracion de sitios, DNS, correo y servicios del VPS DevUs.

La bitacora principal esta en:

```text
MIGRACION_DEVUS_VPS.md
```

Regla de trabajo:

1. Leer `MIGRACION_DEVUS_VPS.md` antes de modificar el VPS, Virtualmin, DNS, correo, WordPress, Node.js u otros servicios relacionados.
2. Actualizar `MIGRACION_DEVUS_VPS.md` despues de cualquier cambio relevante.
3. No subir backups `.tar`, credenciales, llaves privadas ni secretos.
